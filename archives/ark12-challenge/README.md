## Recovery Patterns Discovered

### Empty Output → RETRY
- **Rate:** ~10-15% of Qwen sub-agents return empty on first attempt
- **Success:** 8/8 RETRYs successful (100%)
- **Pattern:** Explicit instructions + full context → consistent delivery

### Meta-Commentary → Trust the File
- **Pattern:** Kimi returns "Let me refine" in completion event
- **Reality:** File already correct
- **Lesson:** Trust the file, not the completion description

### Wrong Path → Move
- **Pattern:** Sub-agents sometimes write to incorrect paths
- **Recovery:** Find, move, verify
- **Lesson:** Context understanding varies by model

The proof was in the 6 days of adversarial testing. The pattern held.
