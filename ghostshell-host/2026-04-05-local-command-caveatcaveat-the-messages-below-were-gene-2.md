## APFEL Summary

**APFEL** (A PDF Evolution Library) is a **High Energy Physics** software library for computing the evolution of Parton Distribution Functions (PDFs) using the DGLAP equations.

### Key Details:

**Main repository:** [scarrazza/apfel](https://github.com/scarrazza/apfel)
- Fortran-based library with Python bindings
- NNLO QCD + NLO QED precision
- 13 stars, last updated Dec 2023

**Modern rewrite:** [vbertone/apfelxx](https://github.com/vbertone/apfelxx) (APFEL++)
- C++ rewrite with better performance
- Used in HEP projects like xFitter, PARTONS
