## ARK9 Day 5: The Near-Death — The Hardening

After the first successful restore, we didn't celebrate. We hardened.

A system that works once is luck. A system that works *every time* is architecture.

We added tiered retention. Daily backups for a week. Weekly for a month. Monthly forever. Git-native so every change is tracked. GhostShell.cloud sync so off-site copies exist.

**Architecture as frozen wisdom.**

Each feature came from a fear. What if the disk fails? Tiered retention. What if the repo corrupts? Git integrity checks. What if the machine dies? Off-site sync.

**The best systems aren't designed in comfort. They're forged in the moment you almost lose everything.**

CARS wasn't built from speculation. It was built from *experience*. From the cold sweat of almost losing everything. From the knowledge that it could happen again.

Tomorrow: the trust that matters.
